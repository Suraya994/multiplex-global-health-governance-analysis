# Data

This folder contains processed 100-country research panels used in the multiplex
global health governance analysis.

The core panel has 2,100 country-year observations:

```text
100 countries x 21 years, 2004-2024
```

Key files:

```text
wdi_panel_100countries_2004_2024_long.csv
io_network_metrics_100countries_2004_2024.csv
io_analysis_panel_100countries_2004_2024.csv
io_analysis_panel_with_iv_100countries_2004_2024.csv
iv_gravity_cepii_100countries.csv
```

Before making the repository public, verify that all processed datasets can be
shared under the relevant source terms.
